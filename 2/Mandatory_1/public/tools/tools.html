Tools for Node:

Express:

Express is a web application framework that enables the support for building software with JavaScript on the server side.

With express you are able to publish the app made with Node.Js to a website.

Nodemon:

Nodemon is a tool used in addition to express if you want a smooth and fast developing process.

Nodemon is a utility that will register and monitor changes made to the source code and automatically restarts the server each time a change has been made.

- It can be implemented using the command: npm install nodemon. It will automatically be set in the package.json file.

- After installation you can simply type in nodemon app.js to start the application, and every time a change has been made to the source code it will restart the app.

Budo:

When you develop a web application you also have the frontend part of your project you need to focus on.

Budo can help you with a fast wisualizing of the frontend part by "Live-reloading" your html pages in the prototype-fase.

- Install budo globally by using the command: npm install budo -g.

- Next, is to navigate to the project folder in the terminal.

- From there, you can type in the command: npx budo 'filename' --live --open.

--live, enables the "Live-reloading". --open, opens a browser with the corresponding html file.
