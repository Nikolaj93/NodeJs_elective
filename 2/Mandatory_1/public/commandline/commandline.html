Navigation in command line:

When navigating folders and accessing different files we use commands such as 'cd' & 'ls'.

'cd' for changing directory and 'ls'(list files), for seeing a list of files in the current folder.

NPM:

NPM stands for Node Package Manager which mean that you have to install NodeJS to take advantage of NPM.

NPM is a package manager and lets developers download different packages for different usages.

Packages are defined in a package.json file and are written in JSON.

To install packages with NPM type: "npm install 'what package to install'.

If I would like to install Express I would type: "npm install express" in command line.
